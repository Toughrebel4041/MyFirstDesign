Website 101

About the website:

Website ini memiliki rantai sebanyak 3 halaman web, yang terdiri dari Halaman Biodata Author, Hobi Author, dan Halaman Informasi Website. Pada laman Profile Author dan Hobby Author, Author mendaur ulang proyek Assesment Pertama pada bulan oktober yang kemudian dipercantik dan menambahkan ketentuan-ketentuan yang diperlukan untuk memenuhi kriteria Penilaian UAS Proyek Bahasa Markup.

Pada semua website ini diaplikasikan Bootstrap 5 (Baca Tentang Bootstrap 5 Disini!), dikarenakan penggunaan bootstrap adalah salah satu keperluan yang perlu dituntaskan untuk memenuhi kriteria penilaian. Selain penggunaan Bootstrap, website ini juga mengaplikasikan penggunaan CSS yang merata di semuanya (Baca tentang CSS Disini!). Sebenarnya penggunaan CSS tidak terlalu diperlukan apabila telah mengaplikasikan Bootstrap, tetapi Author lebih nyaman menggunakan CSS karena lebih terbiasa, penggunaan Bootstrap pada website ini juga sedikit terbatas karena banyak pekerjaan Design yang diambil alih oleh CSS.

Palet warna yang digunakan berorientasi pada Warna Biru tua, Biru Langit, Abu-Abu, dan juga Khaki sebagai warna elemen penambah. Design dari ketiga laman disamakan agar ketiganya memiliki keseragaman design, dimana semua design dan pemilihan warna dilakukan sendiri oleh Author.
